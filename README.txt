Para inicializar o programa servidor, digitar:
python3 server.py <porta>

Para inicializar o programa cliente, digitar:
python3 client.py <ip/endereço> <porta>

O programa servidor funciona sem nenhuma operação externa.

O programa cliente requer entradas do teclado, que são detalhadas no momento em que são requeridas
